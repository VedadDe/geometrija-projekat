Algoritam provjere praznog trougla

Za zadati skup tačaka u ravni i trougao u ravni provjeriti da li postoji ijedna tačka od zadatih da se nalazi u unutrašnjosti trougla.

Varijable:

  • canvas: ElementRef objekat koji predstavlja HTML canvas element.
  • ctx: 2D kontekst za crtanje na canvasu.
  • trokutTacke: Niz koordinata vrhova trokuta.
  • nasumicneTacke: Niz nasumičnih tačaka koje će biti generisane.

Funkcije:

  • ngOnInit(): void
    Ulaz: Nema ulaznih parametara.
    Izlaz: Nema povratne vrijednosti.
    Inicijalizuje komponentu, postavlja 2D kontekst i osluškuje događaj klika na canvasu.
  • obradiKlikNaPlatnu(event: MouseEvent): void
    Ulaz: event: MouseEvent - Dogаđaj klika na platnu.
    Izlaz: Nema povratne vrijednosti.
    Obrada klika na platnu, dodaje tačke u niz trokutTacke i crta trokut kada su sva tri vrha postavljena.
  • generisiNasumicneTacke(): void
    Ulaz: Nema ulaznih parametara.
    Izlaz: Nema povratne vrijednosti.
    Generiše 100000 nasumičnih tačaka i provjerava da li se nalaze unutar trokuta ili ne. Boji tačke u crveno ako se nalaze unutar trokuta, a u plavo ako se nalaze izvan.
  • daLiJeTackaUnutarTrokuta(x: number, y: number): boolean
    Ulaz: x: number - x koordinata tačke, y: number - y koordinata tačke.
    Izlaz: boolean - Vraća true ako je tačka unutar trokuta, false ako nije.
    provjerava da li se tačka nalazi unutar trokuta.
  • povrsinaTrokuta([x1, y1]: [number, number], [x2, y2]: [number, number], [x3, y3]: [number, number]): number
    Ulaz: [x1, y1]: [number, number] - prva tačka, [x2, y2]: [number, number] - druga tačka, [x3, y3]: [number, number] - treća tačka.
    Izlaz: number - Površina trokuta.
    Izračunava površinu trokuta.
  • crtajTacku(x: number, y: number, boja: string = 'plava'): void
    Ulaz: x: number - x koordinata tačke, y: number - y koordinata tačke, boja: string - boja tačke (opciono, podrazumijevana vrijednost 'plava').
    Izlaz: Nema povratne vrijednosti.
    Crta tačku na platnu sa zadatim koordinatama i bojom.
  • crtajTrokut(): void
    Ulaz: Nema ulaznih parametara.
    Izlaz: Nema povratne vrijednosti.
    Crtanje trokuta na platnu.
  • ocisticanvas(): void
    Ulaz: Nema ulaznih parametara.
    Izlaz: Nema povratne vrijednosti.
    Briše sve tačke i trokut sa platna.

Glavna logika:

Korisnik kreira trokut klikom na canvas. Nakon što su postavljeni svi vrhovi, korisnik može generisati nasumične tačke. Tačke unutar trokuta će biti obojene crveno, dok će tačke izvan trokuta biti obojene plavo. Korisnik može očistiti canvas i početi iznova kada god želi.
